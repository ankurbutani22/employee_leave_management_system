# Admin Leave Management (scaffold)

Simple admin-side leave management scaffold using Vite + React + Tailwind.

Run locally:

```powershell
cd admin
npm install
npm run dev
```

Routes:
- `/login` - login page
- `/employees` - all employees list
- `/leaves` - all leaves (approve/cancel)
- `/add-employee` - add employee
- `/my-profile` - admin profile
